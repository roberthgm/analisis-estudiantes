# Student Performance Analysis

Exploratory Data Analysis (EDA) of student academic performance.

## Analysis includes
- Correlation between attendance and final grade (r=0.95)
- Comparison between partial and final grades
- Distribution of grades and attendance
- Identification of outliers

## Technologies
- Python 3.11
- NumPy, Pandas, Matplotlib
- Jupyter Lab

## Key findings
- Strong positive correlation (0.95) between attendance and grades
- 80% of students improved from partial to final exam
- Students with higher attendance consistently scored better
